**Outcome:** Deployed to [clinic-demo.skyshine.online](https://clinic-demo.skyshine.online) with GitHub Actions CI for lint, typecheck, production build, smoke checks, Docker image validation, Dependabot updates, safe `.env.example` defaults, and structured booking API logs.
